Supercell ID liberado no Brasil para o Clash Royale

Recurso que facilita e da mais segurança para contas de jogadores, finalmente é liberado no Clash Royale para Brasileiros!

O recurso Supercell ID já estava funcionando no Brasil, mas apenas para o outro jogo de sucesso da Supercell, o Clash of Clans! E como a empresa já havia prometido, o recurso também seria implementado nesta atualização, demorou um pouco mas finalmente ele foi liberado para o Clash Royale! Se você estava por fora do que se trata, aqui vai uma explicação do que é o Supercell ID:

Com o serviço do Supercell ID, você manterá sua conta de jogo protegida e poderá jogar com ela em todos os seus dispositivos móveis. Para alternar entre contas de jogo diferentes com segurança, você deverá vincular seu Supercell ID a cada uma delas. Depois, você poderá selecionar a opção “Salvar minhas credenciais neste dispositivo” para alternar tais contas com um simples toque. E você não precisará dos códigos de verificação! Para criar o seu Supercell ID, você deverá acessará as configurações do jogo. Depois, tocar no botão correspondente ao “Supercell ID” e seguir as instruções. Para salvar o progresso de outro jogo da Supercell no mesmo Supercell ID, basta iniciar uma sessão em tal jogo com o e-mail do Supercell ID.
